🎯ゴール

本コース終了後、受講者が以下を実施できる状態を目指します。

  • Claude Code の基本操作・コマンド・使い方を理解し、日常業務に導入できる
  • [1] Claude Code を使って ソースコードの解析 を実行できる
  • [2] Claude Code を使って ソフトウェア開発 を実行できる（設定ファイル・Skills/プラグイン・skill-creator・Hooks・MCP・SubAgent を活用）
  • [3] Claude Code を使って チーム開発 を実行できる
